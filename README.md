# 🛤 전국 팔도
## 웹페이지 소개
전국 각지의 특산물을 소개하는 '전국팔도' 입니다.

이 웹페이지는 COVID-19로 인해 경제적인 피해를 입은 농가에게 도움이 되고자 하는 취지에서 제작되었습니다.

1. 각 지역의 여러 특산물을 소개합니다.
2. 특산물의 산지를 지도에 표시해 한 눈에 알아볼 수 있도록 나타냅니다.
3. 해당 특산물의 구매를 원하는 경우 구매사이트로 연결합니다.
  
## 개발환경

+ JavaScript
